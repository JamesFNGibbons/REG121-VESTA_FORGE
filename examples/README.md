# Starter component library (template)

This folder is a **copy-paste template**. Your real component library should live **outside** the REG121 vectoriser repository.

```bash
cp -R examples/component-library-starter ~/my-reg121-library
./121 library configure   # point at ~/my-reg121-library
```

Then edit HTML and `catalogue.py` in that external directory.

See [README.md](README.md) inside this folder for HyperUI attribution and MIT terms.
